# СинхроЛинк - SaaS Landing Page (RU)

Русскоязычный лендинг для SaaS-платформы коммуникационной аналитики.

## Tags

`saas` `light` `general` `pricing` `testimonials`

## Описание

СинхроЛинк - это шаблон лендинга для SaaS-продукта, ориентированного на анализ корпоративных коммуникаций. Включает все необходимые секции для продвижения B2B SaaS решения.

## Секции

- **Навигация** - фиксированная шапка с адаптивным мобильным меню
- **Hero** - главный блок с призывом к действию
- **Статистика** - ключевые метрики и показатели
- **Кейсы** - карусель историй успеха клиентов
- **Интеграции** - анимированная карусель партнерских логотипов
- **Тарифы** - сравнительная таблица тарифных планов
- **FAQ** - аккордеон часто задаваемых вопросов
- **Футер** - навигация и социальные ссылки

## Технологии

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Radix UI
- Lucide Icons

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Особенности

- Полная русская локализация
- Кириллический шрифт Inter с поддержкой всех начертаний
- Анимации на Framer Motion
- Адаптивный дизайн
- Поддержка light/dark темы
- SEO-оптимизация
